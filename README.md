# Power BI Sales & Performance Dashboard

## Project Overview
This project is an interactive Power BI dashboard developed to analyze business performance, track KPIs, and generate actionable insights from operational and sales data.

The dashboard helps monitor trends, compare performance metrics, and support data-driven decision-making through visual analytics.


## Tools & Technologies Used
- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel
- Data Modeling
- Data Visualization

## Key Features
- KPI Tracking Dashboard
- Interactive Visualizations
- Month-over-Month Performance Analysis
- Sales & Revenue Analysis
- Dynamic Filters and Slicers
- Trend & Variance Analysis
- Data Cleaning and Transformation using Power Query

## DAX Measures Used
- Total Sales
- Profit Margin
- Month-over-Month Growth
- KPI Comparison
- Percentage Contribution
- Variance Analysis


## Insights Generated
- Identified top-performing regions/products
- Analyzed sales trends and business growth
- Compared actual performance against targets
- Highlighted performance variations and patterns
- Improved reporting clarity through interactive visuals
